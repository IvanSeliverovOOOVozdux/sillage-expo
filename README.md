# SILLAGE — лендинг выставки нишевой парфюмерии

**Шаблон портфолио #3** · тёмный luxury · one-page · русский язык

Одностраничный промо-лендинг для (вымышленной) международной выставки нишевой
парфюмерии **SILLAGE** (Москва, Центральный Манеж, 13–17 ноября 2026).
Тёмная «люксовая» эстетика: near-black фон + шампань-золото + serif-типографика.

---

## Стек

- **Чистый статичный HTML/CSS/JS в одном файле** — `index.html`. Сборки нет.
- Шрифты: Google Fonts — **Prata** (serif, заголовки) + **Manrope** (sans, текст).
- Картинки: Unsplash (CDN) с автоматическим откатом на `picsum.photos` (`onerror`).
- Ноль зависимостей, ноль npm. Открывается двойным кликом по `index.html`.

## Как запустить локально

Просто открой `index.html` в браузере. Или подними любой статик-сервер:

```bash
npx serve .          # или: python -m http.server
```

## Деплой на Vercel

Это статика, билд не нужен.

**Вариант А — через GitHub (рекомендуется):**
1. Запушь репозиторий на GitHub.
2. На vercel.com → **Add New → Project** → импортируй репозиторий.
3. Framework Preset: **Other**. Build Command: пусто. Output Directory: `./` (корень).
4. Deploy. Дальше — автодеплой при каждом push.

**Вариант Б — без гита:** `npm i -g vercel` → `vercel` в папке проекта.

## Структура секций

Хедер (фиксированный) → Hero → О выставке → Архитектура аромата (пирамида нот) →
Участники → Инсталляции (галерея) → Билеты → Регистрация → Футер.

## Кастомизация под нового клиента

### 1. Палитра — CSS-переменные в `:root` (вверху `<style>`)
```css
--bg / --bg2      /* фон (near-black) */
--gold / --gold-soft / --gold-deep   /* акцент — поменять для другого бренда */
--text / --muted / --muted-2         /* текст */
--line / --line-soft                 /* бордеры */
```
Сменив `--gold` и `--gold-soft`, получаешь другой акцент (производные свечения фона
и бордеры завязаны на них).

### 2. Шрифты
Ссылка Google Fonts в `<head>` + `font-family` в `:root`/правилах. Заголовки — Prata,
текст — Manrope. Обе с кириллицей.

### 3. Контент
Текст правится прямо в разметке секций. Ключевые места: название/даты/город в Hero,
заголовки секций, список участников (`.house`), тарифы билетов (`.ticket`), контакты в футере.

### 4. Изображения
- Галерея и коллаж «О выставке» — ссылки `images.unsplash.com/...` с `data`-фолбэком на
  `picsum.photos`. Для прода заменить на реальные фото клиента.
- Флакон в Hero — инлайновый **SVG** (не растр), редактируется в разметке.

### 5. Интерактив (всё в `<script>` внизу)
- **Распыление парфюма** (`emitSpray`/`drawSpray`) — canvas-частицы с аддитивным свечением,
  3 типа (морось/капли/туман). Триггер: первый скролл вниз + клик по флакону; сброс наверху.
  Тюнинг: кол-во частиц в циклах `for`, скорость `sp`, угол `SPREAD`, трение `*0.982`,
  гравитация `grav`, прозрачность `alpha`, точка сопла `ox/oy`.
- **Пирамида нот** (`TIERS` + `setTier`) — клик/наведение меняет активный слой.
- **Счётчики** (`data-count`) — анимация от 0 при появлении.
- **Reveal** — `IntersectionObserver` по элементам `.reveal` (каскад через `data-i`).
- **Форма регистрации** — кастомный селект + валидация (имя/e-mail/тариф), имитация отправки.

## Доступность / адаптив

- Учитывается `prefers-reduced-motion` (анимации отключаются).
- Раскладка адаптируется: брейкпоинты 900px и 560px (hero/about/pyramid → 1 колонка,
  галерея → 2, билеты → стопкой, мобильное меню-бургер).
